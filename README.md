# Eviction Defense Demo

Static client demo with three responsive one-page UI directions for an eviction document coordination service.

- Cal-inspired: scheduling-first, crisp, operational intake flow
- Pinterest-inspired: warm, visual, card-based service browsing
- Superhuman-inspired: premium, minimal, process-led presentation

Use the root URL or `#cal` for the Cal-inspired default, `#pinterest` for the Pinterest-inspired version, and `#superhuman` for the Superhuman-inspired version. Each version includes a dark-mode toggle.

Content is paraphrased from the requested public reference sites and presented as a demo concept. This is a document coordination service concept, not a law firm.
